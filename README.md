A business analytics project focused on improving donor engagement by analysing touchpoints, channel effectiveness, donation timing, and supporter value.

**Skills used:** Power BI, DAX, business analysis, segmentation, campaign analytics

### 2. Cybersecurity Analytics Dashboard
A Power BI dashboard solution designed to monitor phishing risk, employee behaviour, and training effectiveness across departments and branches.

**Skills used:** Power BI, KPI design, dashboarding, stakeholder reporting

### 3. Customer Analytics & Segmentation Using SQL
A SQL project focused on customer behaviour analysis, loyalty performance, RFM segmentation, and marketing effectiveness.

**Skills used:** SQL, JOINs, CTEs, aggregation, customer segmentation, business strategy

A time-series forecasting project using deep learning models to predict PM2.5 levels and improve short-term environmental forecasting accuracy.

**Skills used:** Python, TensorFlow, GRU, LSTM, time-series forecasting

A machine learning regression project to predict solar energy output using weather and solar geometry variables.

**Skills used:** Python, TensorFlow, regression modelling, neural networks
